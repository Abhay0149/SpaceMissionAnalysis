## 🌌 Project Overview

**Space Mission Analytics** is a data analytics and machine learning project that analyzes historical space mission datasets to discover patterns and insights.

The project explores launch trends, mission outcomes, and predicts mission success using a machine learning model.

It also includes a simple **interactive dashboard** to visualize mission analytics.

---

## ✨ Features

* 🚀 Space mission dataset analysis
* 📊 Interactive analytics dashboard
* 🤖 Machine learning mission success prediction
* 📈 Data visualization of mission trends
* 🌐 Clean frontend interface

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/Abhay0149/SpaceMissionAnalysis.git
```

Go to project folder

```
cd SpaceMissionAnalysis
```

Install dependencies

```
pip install -r requirements.txt
```

Run backend

```
python backend/app.py
```

Open **index.html** in your browser.
